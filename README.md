## Getting Started
### Prerequisites
- Node.js
- npm
- Expo CLI

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/your-expo-project.git
   cd your-expo-project
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   expo start
   ```
